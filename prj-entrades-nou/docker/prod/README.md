# Docker — producció

Utilitza imatges construïdes des de `docker/dockerfiles/` sense muntar el codi com a volum d’edició. Omple secrets via variables d’entorn o fitxers segurs fora del repositori.

El fitxer `docker-compose.yml` d’aquesta carpeta es pot afegir quan es defineixi el desplegament (Speckit Phase 1 es centra en `docker/dev/`).
