# Especificació d’aquest canvi

El **contingut normatiu** dels requisits (A–E, temps real, rols) està a:

**`openspec/specs/admin-platform-suite/spec.md`**

Aquest fitxer existeix per mantenir l’estructura `openspec/changes/<change>/specs/<capability>/spec.md` i enllaçar el canvi **admin-platform-suite** amb la capacitat homònima.

Durant la implementació, qualsevol **delta** respecte a l’especificació “main” s’ha de reflectir primer al document principal anterior i després es pot resumir aquí com a nota de versió (opcional).

**Flux d’entrega (Git, proves, merge)**: vegeu `openspec/changes/admin-platform-suite/WORKFLOW.md`.
